# asg-plugins-meta-framework

ASG共通行動原則 plugin。全agentに配布する共通スキル集。

## Skills

| スキル | 概要 |
|---|---|
| `self-restraint` | 責務外スキル・MCP 自己抑制ルール |
| `coding-discipline` | ASG実装・生成時の行動原則 v1 |
| `handoff-format` | handoff作成ルール |

## Install

```
/plugin install asg-meta-framework@asg-marketplace
```

## Maintenance

- `self-restraint`: `scripts/shared-rules/self-restraint.md` と同期（並行運用期間: 4/25-5/2）
- `coding-discipline`: `scripts/shared-rules/coding-discipline.md` と同期
- `handoff-format`: 本リポ独自管理
